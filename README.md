# Практическая работа №2

## Разработка TCP эхо-сервера

---

### 📌 Цель работы

Изучить основы сетевого взаимодействия и реализовать клиент-серверное приложение на основе TCP-сокетов.

---

### ⚙️ Описание проекта

В рамках работы реализованы:

* TCP-сервер, принимающий подключения клиентов
* TCP-клиент для отправки сообщений серверу
* механизм обмена сообщениями (эхо-сервер)
* логирование работы клиента и сервера

---

### 🧠 Используемые технологии

* Python 3
* socket
* logging

---

### 🚀 Запуск проекта

#### 1. Запуск сервера:

```id="k4y6pa"
python server.py
```

---

#### 2. Запуск клиента:

```id="6qg8ds"
python client.py
```

---

### 🔄 Принцип работы

1. Клиент подключается к серверу
2. Пользователь вводит сообщение
3. Клиент отправляет сообщение серверу
4. Сервер:

   * принимает сообщение
   * обрабатывает его
   * отправляет ответ обратно
5. Клиент получает ответ и выводит его

---

### 🔧 Особенности реализации

✔ Сервер работает по TCP протоколу
✔ Обработка сообщений в цикле
✔ Поддержка команды `exit` для завершения соединения
✔ Логирование всех операций

---

### 🧪 Модификация сообщения

Сервер изменяет сообщение перед отправкой обратно:

```id="5n4k2z"
message.upper()[::-1]
```

👉 Это означает:

* перевод в ВЕРХНИЙ РЕГИСТР
* разворот строки

Пример:

```id="c4v9lm"
hello → OLLEH
```
